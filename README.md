# Canyon
## Purpose
This is a side project where I explore the inner workings of Object Oriented languages.
## Description
```core.c``` and ```core.h``` contain partial implementations of Object, Integer, and String classes in C.
## Current Objective(s)
- Implement a set of basic classes in C using structs
## Future Goals
- Establish CanyonLang syntax
- Write a program to transpile CanyonLang to C
- Create a full set of builtin classes for Canyon
- Make some utility libraries in CanyonLang
- Write a dedicated compiler for CanyonLang
